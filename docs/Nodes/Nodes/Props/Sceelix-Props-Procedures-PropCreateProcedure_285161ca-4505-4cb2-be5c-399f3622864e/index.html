Prop Create

Creates Sceelix Props, i.e. entities that only exist in the Sceelix Designer to build a surrounding environment.

Tags: Fire, Ocean

Parameters​

Primitive [Multi-Type List]
List of props to create.
Fire [Compound] [Adds ports 1]
Creates an fire prop at the origin.
Ocean [Compound] [Adds ports 2]
Creates an ocean plane at a given height, with infinite extents.
Water Level [Float]
Height of the water level.
Water Color [Color]
Color of the ocean water.
Wave Scale [Float]
Scale of the ocean waves.

Inputs​

This node has no native inputs.

Outputs​

This node has no native outputs.

Parameter Outputs​

  • [1] Output [Actor]: The fire prop that was created.
  • [2] Output [Entity]: The ocean prop that was created.
