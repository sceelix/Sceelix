Mesh Flip

Flips mesh faces or coordinates.

Tags: Mesh, Coordinates, Faces

Parameters​

Type [Select]
Type of mesh flipping operation to perform.
Coordinates [Compound]
Flips coordinates of all vertex positions.
Coordinates [Choice]
Coordinates to flip. For example, "X<->Y" flips the X with the Y coordinate.
Faces [Compound]
Flips the orientation of the mesh faces.

Inputs​

  • Input [Single | Mesh]: The Mesh to process.

Outputs​

  • Output [Mesh]: The Mesh that was processed.
