Billboard Create

Creates billboard entities with given characteristics.

Tags: Billboard

Parameters#

Color [Color]
Color tint of the billboard.
Image [File]
Image to be set as billboard.
Size [Vector2D]
Size of the billboard in width (X) and height (Y).

Inputs#

This node has no native inputs.

Outputs#

  • Output [Billboard]: Billboard created according to the defined parameters and/or inputs.
